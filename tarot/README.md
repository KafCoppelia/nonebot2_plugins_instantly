## 安装

1. `git clone`；

2. 将`tarot`文件夹加入你的`bot/plugins`下；

3. 占卜一下你的今日运势！

## 功能

1. 塔罗牌！占卜今日运势；
2. 可群聊或私聊占卜；
3. *TODO* 目前群聊内占卜不启用转发功能塔罗牌CQ码不解析的bug；

## 命令

塔罗牌、占卜、抽卡、今日运势
