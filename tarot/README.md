## 安装

1. `git clone`；

2. 将`tarot`文件夹加入你的`bot/plugins`下；`resource`内塔罗牌资源放置于`./data/tarot`下；

3. *可选：在`env`内配置`CHAIN_REPLY`、`TAROT_PATH`：

```python
CHAIN_REPLY=false               # 是否启用转发模式
TAROT_PATH="./data/tarot_card"  # 塔罗牌储存路径，默认位于./data/tarot_card
```

4. 占卜一下你的今日运势！

## 功能

1. 塔罗牌！占卜今日运势；
2. 可群聊或私聊占卜；

## 命令

塔罗牌、占卜
