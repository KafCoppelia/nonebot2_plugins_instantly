## 安装

1. `git clone`；

2. 将`crazy_thursday`文件夹加入你的`bot/plugins`下；

3. *可选：在`env`内指定读取的文案信息`post.json`：

```python
CRAZY_PATH="./data/crazy_thursday"
```

默认读取数据位置：

```
"./data/crazy_thursday"
```

## 命令

1. [疯狂/狂乱]星期[一|二|三|四|五|六|日]；
2. 输入**疯狂星期八**等不合法时间将提示；
