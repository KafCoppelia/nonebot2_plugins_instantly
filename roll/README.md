## 安装

1. `git clone`；

2. 将`roll`文件夹加入你的`bot/plugins`下；

3. Enjoy:tada:

## 功能

掷骰！扔出指定个数的多面骰子。

## 命令

rd、roll、掷骰 + <x>d<y>， x指定个数，y指骰子面数。
